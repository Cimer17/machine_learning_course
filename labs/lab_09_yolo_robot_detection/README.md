# Лабораторная работа №9. Обнаружение объектов с YOLO

Цель работы: применить подход YOLO для разметки и обнаружения объектов.

## Что изучается

- классы объектов для мобильных роботов;
- формат YOLO-разметки;
- координаты bounding box;
- структура датасета `images/train` и `labels/train`;
- подготовка YAML-файла датасета.

## Материалы

- `YOLO.ipynb` - практический ноутбук.
- `verified_materials/assets/mobile_robots_lab.jpg` - исходное изображение.
- `verified_materials/outputs/yolo_robot_classes_annotated.jpg` - изображение с нанесенной разметкой.
- `verified_materials/robot_yolo_dataset/` - минимальный датасет в формате YOLO.

Мини-датасет нужен для проверки структуры и формата. Для полноценного обучения его следует расширить большим количеством размеченных изображений.
