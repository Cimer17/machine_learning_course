# Лабораторная работа №3. Циклы и условные инструкции

Цель работы: познакомиться с условными инструкциями и циклами, понять их структуру и логику работы.

## Что изучается

- `if`, `elif`, `else`;
- логические выражения;
- циклы `for` и `while`;
- вложенные условия и циклы;
- решение задач по вариантам.

## Материалы

Отдельных файлов к этой работе в исходных материалах не было. Рекомендуемый формат выполнения: создать ноутбук с решениями, примерами входных данных и выводом результатов.
