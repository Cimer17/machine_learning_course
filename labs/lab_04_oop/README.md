# Лабораторная работа №4. Основы объектно-ориентированного программирования

Цель работы: научиться писать код в парадигме ООП.

## Что изучается

- классы и объекты;
- поля и методы;
- конструктор `__init__`;
- инкапсуляция данных;
- простое моделирование предметной области.

## Материалы

Отдельных файлов к этой работе в исходных материалах не было. Для курса по робототехнике удобно выполнять задания на примерах классов роботов, датчиков, приводов или учебных соревнований.
