# Лабораторная работа №7. Детекция монет с OpenCV

Цель работы: реализовать детекцию круглых объектов на изображении с использованием методов компьютерного зрения OpenCV.

## Что изучается

- загрузка изображения через OpenCV;
- перевод изображения в оттенки серого;
- сглаживание и выделение границ;
- поиск окружностей и визуализация результата;
- сохранение промежуточных изображений.

## Материалы

- `OpenCV.ipynb` - практический ноутбук.
- `verified_materials/assets/coins_real.jpg` - исходная фотография монет.
- `verified_materials/assets/extra_coin_images/` - дополнительные изображения монет для проверки алгоритма в разных условиях.
- `verified_materials/outputs/` - проверенные изображения после обработки.

Откройте ноутбук из этой папки, чтобы относительные пути к `verified_materials` работали без дополнительных настроек.

## Дополнительные ассеты

В папке `extra_coin_images` лежат варианты исходного изображения:

- `coins_01_rotated.jpg` - поворот камеры.
- `coins_02_low_light.jpg` - пониженная освещенность.
- `coins_03_high_contrast.jpg` - повышенный контраст.
- `coins_04_noise.jpg` - шум на изображении.
- `coins_05_soft_blur.jpg` - небольшое размытие.
- `coins_06_perspective.jpg` - перспективное искажение.

Файл `manifest.csv` кратко описывает назначение каждого изображения. Превью всех дополнительных ассетов сохранено в `verified_materials/outputs/extra_coin_images_contact_sheet.jpg`.
