# Лабораторная работа №6. Прогнозирование

Цель работы: создать простейшую нейронную сеть и получить прогноз.

## Что изучается

- однонейронная модель `predict(P, w) = P * w`;
- ошибка прогноза;
- одна итерация обновления веса;
- влияние скорости обучения;
- визуализация прогноза до и после обучения.

## Материалы

Отдельных файлов к этой работе в исходных материалах не было. В качестве учебного сценария используется прогноз вероятности победы мобильного робота по средней скорости на трассе.
