# Лабораторная работа №10. Детектирование человека или робота в реальном времени

Цель работы: применить предобученную нейронную сеть YOLO для обнаружения объектов в видеопотоке с веб-камеры.

## Что изучается

- установка `ultralytics` и `opencv-python`;
- загрузка предобученной модели YOLO;
- чтение кадров с веб-камеры;
- запуск детекции на каждом кадре;
- отображение результата в реальном времени.

## Материалы

- `models/yolov8n.pt` - веса компактной YOLOv8-модели из материалов курса.

Для запуска понадобится веб-камера. Если модель не найдена локально, `ultralytics` обычно может загрузить ее автоматически при наличии доступа к сети.
