# Лабораторная работа №2. Базовые функции и типы данных Python

Цель работы: научиться вводить и выводить данные, создавать переменные и выполнять арифметические операции.

## Что изучается

- `print()` и `input()`;
- переменные и базовые типы данных;
- преобразование типов;
- арифметические операторы;
- простые вычислительные программы.

## Материалы

Отдельных файлов к этой работе в исходных материалах не было. Рекомендуемый формат выполнения: создать новый `.ipynb` в этой папке и сохранить решения заданий по вариантам.
