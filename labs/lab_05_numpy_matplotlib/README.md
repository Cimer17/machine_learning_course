# Лабораторная работа №5. Библиотеки Matplotlib и NumPy

Цель работы: освоить создание изображений и графиков средствами Matplotlib и научиться работать с массивами NumPy.

## Что изучается

- массивы `numpy.ndarray`;
- индексация, срезы и векторные операции;
- генерация числовых данных;
- линейные графики, гистограммы и диаграммы;
- оформление графиков: подписи, легенды, сетка.

## Материалы

- `st.csv` - большой CSV-файл с параметрами мобильных роботов по примеру из пособия: `robot`, `mass`, `platform`, `speed` и дополнительные признаки.
- `plot_st_csv.py` - пример построения графика «масса — скорость» по данным из `st.csv`, аналогично рисунку 42.

Рекомендуемый формат выполнения: создать ноутбук с вычислениями, графиками и краткими выводами по каждому заданию.
