# Лабораторная работа №8. Библиотека Keras

Цель работы: реализовать модель классификации изображений с использованием TensorFlow/Keras.

## Что изучается

- подготовка изображений для обучения;
- формирование обучающей и проверочной выборок;
- построение простой CNN-модели;
- обучение модели и анализ графика качества;
- проверка предсказаний на изображениях мобильных роботов.

## Материалы

- `Keras.ipynb` - практический ноутбук по классификации типов мобильных роботов.
- `verified_materials/mobile_robot_dataset/` - готовый датасет для Keras в формате `train/validation/test/<class_name>/image.jpg`.
- `verified_materials/assets/mobile_robots_lab.jpg` - исходная фотография лаборатории.
- `verified_materials/assets/robot_crops/` - базовые кропы роботов по классам.
- `verified_materials/assets/robot_augmented/` - расширенный набор изображений с поворотами, отражением и изменением освещенности.
- `verified_materials/assets/robot_assets_manifest.csv` - описание классов и происхождения файлов.
- `verified_materials/outputs/` - кропы, график обучения и примеры предсказаний.

Ноутбук использует тематику курса по мобильной робототехнике и заменяет абстрактные датасеты на изображения роботов из материалов курса.

## Классы ассетов

- `wheeled_robot` - колесные роботы.
- `tracked_robot` - гусеничные роботы.
- `legged_robot` - шагающий робот.

Всего добавлено 8 базовых кропов и 90 аугментированных изображений: по 30 изображений на каждый класс. Из них собран Keras-датасет на 90 изображений: 63 для обучения, 18 для проверки и 9 для теста. Превью набора сохранено в `verified_materials/outputs/robot_assets_contact_sheet.jpg`.
