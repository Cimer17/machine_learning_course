# Лабораторная работа №1. Установка Python, Jupyter Notebook

Цель работы: познакомиться с языком Python и веб-приложением Jupyter Notebook.

## Что изучается

- установка Python и `pip`;
- запуск терминала и выполнение базовых команд;
- установка и запуск Jupyter Notebook;
- создание первого ноутбука;
- выполнение ячеек с кодом и текстом.

## Материалы

Отдельных файлов к этой работе в исходных материалах не было. Для выполнения достаточно установить окружение из корневого `README.md` и проверить запуск Jupyter Notebook.
