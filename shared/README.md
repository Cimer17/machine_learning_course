# Shared materials

Папка `shared` содержит общие материалы к учебному пособию по главам. Лабораторные работы остаются в `labs/`, а здесь собраны данные, ассеты, примеры и инструкции, которые удобно переиспользовать при подготовке занятий, ноутбуков и методических материалов.

## Разделы

| Глава | Материалы |
| --- | --- |
| 01 | `chapters/01_python_jupyter` - установка Python и Jupyter |
| 02 | `chapters/02_python_basics` - базовые типы и ввод/вывод |
| 03 | `chapters/03_functions_modules` - функции и модули |
| 04 | `chapters/04_matplotlib` - построение графиков |
| 05 | `chapters/05_numpy_csv` - NumPy и CSV, включая `st.csv` |
| 06 | `chapters/06_programming_paradigms` - парадигмы программирования |
| 07 | `chapters/07_oop` - объектно-ориентированное программирование |
| 08 | `chapters/08_recursion` - рекурсия |
| 09 | `chapters/09_neural_networks` - основы нейронных сетей |
| 10 | `chapters/10_opencv` - OpenCV и детекция монет |
| 11 | `chapters/11_keras_mobile_robot` - Keras и датасет мобильных роботов |
| 12 | `chapters/12_yolo` - YOLO и разметка роботов |

## Зеркало пакетов

Папка `package_mirror` содержит зафиксированные версии Python-пакетов, скрипты для скачивания wheel-файлов в локальный `wheelhouse` и инструкцию по установке без доступа к интернету.
