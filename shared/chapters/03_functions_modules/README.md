# Глава 03. Функции и модули

Глава используется для перехода от отдельных команд к переиспользуемому коду.

## Темы

- объявление функций через `def`;
- аргументы и возвращаемые значения;
- импорт стандартных модулей;
- разбиение кода на небольшие функции.

Для примеров удобно брать данные из `../05_numpy_csv/data/st.csv` и писать функции расчета средней скорости, средней массы и фильтрации роботов по типу платформы.
