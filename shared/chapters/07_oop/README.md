# Глава 07. Объектно-ориентированное программирование

Материалы главы удобно связывать с мобильной робототехникой.

## Идеи классов

- `Robot` - базовые параметры: имя, масса, скорость, платформа.
- `Sensor` - тип датчика и дальность измерения.
- `Battery` - емкость и расчет времени работы.
- `RobotFleet` - коллекция роботов и методы фильтрации.

Для наполнения объектов используйте `../05_numpy_csv/data/st.csv`.
