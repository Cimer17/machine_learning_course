# Глава 08. Рекурсия

Глава не требует отдельных ассетов, но хорошо связывается с задачами обхода структур.

## Рекомендуемые задания

- рекурсивный расчет факториала;
- рекурсивный обход вложенного словаря параметров робота;
- поиск файла в дереве папок курса;
- подсчет количества изображений в датасете `../11_keras_mobile_robot/mobile_robot_dataset`.
