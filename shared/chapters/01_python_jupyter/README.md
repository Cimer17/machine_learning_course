# Глава 01. Python и Jupyter Notebook

Материалы главы помогают подготовить рабочее окружение для всего курса.

## Использовать вместе с

- `shared/package_mirror/README.md` - установка зависимостей.
- `requirements.txt` - корневой список пакетов курса.

## Проверка окружения

```powershell
python --version
python -m pip --version
jupyter notebook
```

Для курса рекомендуется Python 3.12. Python 3.14 лучше не использовать для этой версии набора пакетов, потому что библиотеки машинного обучения обычно добавляют поддержку новых версий Python позже.
