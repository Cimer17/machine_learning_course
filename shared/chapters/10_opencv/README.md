# Глава 10. OpenCV

## Материалы

- `assets/coins_real.jpg` - исходное изображение монет.
- `assets/extra_coin_images/` - дополнительные варианты изображения: поворот, слабый свет, контраст, шум, размытие и перспектива.
- `outputs/extra_coin_images_contact_sheet.jpg` - превью набора.

Эти файлы можно использовать для экспериментов с `cv2.imread`, переводом в оттенки серого, фильтрацией, Canny и HoughCircles.
