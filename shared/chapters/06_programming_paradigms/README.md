# Глава 06. Парадигмы программирования

Глава нужна для сравнения процедурного, функционального и объектно-ориентированного подходов.

## Рекомендуемая практика

Возьмите одну задачу из курса, например фильтрацию роботов по типу платформы из `../05_numpy_csv/data/st.csv`, и реализуйте ее тремя способами:

- процедурно: последовательность команд;
- функционально: функции фильтрации и преобразования;
- объектно: класс `Robot` и методы работы с коллекцией.
