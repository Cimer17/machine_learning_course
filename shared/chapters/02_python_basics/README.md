# Глава 02. Базовые функции и типы данных Python

Материалы главы не требуют внешних файлов. Для практики достаточно создать новый ноутбук и выполнить задания на:

- `print()` и `input()`;
- переменные;
- типы `int`, `float`, `str`, `bool`;
- арифметические операции;
- преобразование типов.

Рекомендуемый сюжет для примеров: параметры мобильного робота, скорость движения, масса, время автономной работы.
