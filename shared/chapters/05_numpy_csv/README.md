# Глава 05. NumPy и CSV

## Материалы

- `data/st.csv` - большой файл с параметрами мобильных роботов по примеру из пособия и рисунка 42.
- `examples/plot_st_csv.py` - пример чтения CSV и построения графика «масса — скорость».

Файл `st.csv` сохраняет первые четыре поля из пособия:

```csv
robot,mass,platform,speed
```

Дальше добавлены дополнительные признаки: грузоподъемность, емкость батареи, автономность, максимальный уклон, клиренс, датчики и сценарий применения.
