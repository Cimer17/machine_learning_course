# Курс по Python, нейронным сетям и компьютерному зрению

Репозиторий оформлен как учебный курс для студентов, которые осваивают Python, Jupyter Notebook, основы машинного обучения и практические задачи компьютерного зрения в контексте мобильной робототехники.

Курс начинается с настройки окружения и базового синтаксиса Python, затем переходит к структурам данных, условиям, циклам, объектно-ориентированному программированию, NumPy и Matplotlib. Финальная часть посвящена нейронным сетям и компьютерному зрению: простому прогнозированию, детекции монет через OpenCV, классификации изображений мобильных роботов через Keras и разметке/детекции объектов с YOLO.

## Что нужно для курса

- Python 3.10 или новее.
- Jupyter Notebook или JupyterLab.
- Базовые библиотеки: `numpy`, `matplotlib`, `pillow`.
- Для компьютерного зрения: `opencv-python`.
- Для нейронных сетей: `tensorflow`.
- Для YOLO: `ultralytics`.

Установка зависимостей:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

Запуск конкретной лабораторной лучше выполнять из ее папки, чтобы относительные пути к материалам работали сразу:

```powershell
cd labs\lab_07_opencv_coin_detection
jupyter notebook OpenCV.ipynb
```

## Материалы

Исходные локальные материалы курса (`Материалы/`, `Ноутбуки/`, `_posobie.docx`) добавлены в `.gitignore`, чтобы не хранить черновые и крупные файлы в истории. Проверенные материалы, которые нужны для выполнения лабораторных, разложены по соответствующим папкам `labs/`.

Практические данные приложены к лабораторным 5 и 7-9. В лабораторной 5 есть файл `st.csv` с параметрами мобильных роботов для построения графика «масса — скорость». В лабораторной 7 есть дополнительный набор изображений монет для проверки алгоритма при разном освещении, шуме, повороте и перспективе. В лабораторной 8 есть готовый Keras-датасет мобильных роботов, кропы роботов и сбалансированный набор аугментированных изображений по трем классам.

Для лабораторной 10 добавлены веса `yolov8n.pt`, которые можно использовать с `ultralytics` для запуска детекции в реальном времени.
